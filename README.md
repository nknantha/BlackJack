<h1 align="center">BlackJack</h1>

<p align="center">
A Python implementation of BlackJack with CUI.
</p>

## Usage
```
# To install requirements
python3 -m pip install -r requirements.txt

# To run
python3 BlackJack.py

# On Windows use "python" instead of "python3"
```
